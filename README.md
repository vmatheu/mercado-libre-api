## Description

Esta api busco productos desde la api de mercado libre.

Se puede buscar por frase:

```bash
localhost:3001/api/items/q={frase para buscar}
```

Se puede buscar por Id:

```bash
localhost:3001/api/items/{frase para buscar}
```


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
